## Final Output

**SwiftUI architecture:** iOS 17+, one `@MainActor @Observable` app store as the single source of UI-facing truth, injected at the app root. **Persistence:** Hand-rolled SQLite, not SwiftData, not Core Data, not any third-party wrapper (no GRDB, no SQLite.swift) — a thin (~200-line) first-party wrapper directly over `libsqlite3`. Three tables: projects, decisions (append-only), and a metadata table holding `surfacedProjectID` and `lastResolvedAt` explicitly, so the surfaced slot is persisted state, never a query side effect. Every mutation (Act/Snooze/Release/delete-of-surfaced) is one atomic transaction — clear/set surfaced project, insert decision row, update metadata, all-or-nothing.
