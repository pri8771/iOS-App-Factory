Each lane works off its own long-lived branch — `lane/data-domain`, `lane/services-utilities`, `lane/primary-ui`, `lane/polish-resilience` — rebased onto `main` before starting each new task, never force-pushed over another lane's history. Merge to `main` happens per completed task once its acceptance criteria pass, as a small squashed commit referencing the task id (e.g. `T-013: AnalysisOrchestrator manual path`), in dependency order — a task doesn't merge before the tasks it `depends_on` are already on `main`. `project.pbxproj`/entitlements changes merge first and fast, ahead of the feature code that depends on the new target/file existing.

**Testing responsibilities:** as split above — data_domain (invariant/cascade tests), services_utilities (unit tests + golden-set regression), primary_ui (view-model tests), polish_resilience (all XCUITest, accessibility pass, real-device Foundation Models session, StoreKit fail-closed drills) — matching the go/no-go gates already locked in `project_plan`.

**Communication protocol:** any lane that needs a protocol/DTO signature changed from what's frozen in `tech_specs`' interfaces-json raises it in this room as an explicit amendment before coding around it locally — silent local changes break whoever else is coding against the old signature. Each lane reports task completion against its stated acceptance criteria before a dependent lane starts the task that depends on it. Fakes/stubs for `ScopeExtractor`, `EntitlementService`, `QuotaService`, `ExportService` (built in T-001/T-002 era) mean primary_ui is never blocked waiting on real implementations — it codes against the protocol + stub, and swapping in the real implementation is a DI change, not a rewrite.

**Sequencing gate (non-negotiable, enforced via task dependencies below, not goodwill):** no AI-path work (`AIScopeExtractor`, AI-unavailable-state screens) starts until the manual-only loop (segmenter → manual extractor → orchestrator's manual methods → review → draft → plain-text export) is proven end-to-end on fixtures, per the locked milestone-2 gate.
